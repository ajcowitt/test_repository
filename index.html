Git Commands

  • init creates a new repository
  • clone creates a working copy of a local repository
  • add adds files to the Index
  • commit commits changes to the HEAD
    Use with option -m "Add a description of the changes"
  • push pushes changes in the HEAD of the local working copy to the remote repository.
    Example: git push origin master
  • remote add connects local repository to remote server in format
    git remote add origin
